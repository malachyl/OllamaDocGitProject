# Installing Ollama on Windows
<!-- If you update the document title, ensure you update the title in the metadata above to match -->

To get started with Ollama, you need to download and install the platform on your local machine. For Mac and Linux, you install Ollama from the command line; for Windows you install directly from the download script executable. 

## Before you begin
Before installing Ollama, check that your local machine meets the minimum specifications for installation and operation. For mor information on the specifications for Ollama, see <!-- ToDo - add link once the project is uploaded to the Web>

## Procedure

1. Browse to the Ollama [website](https://ollama.com/).
2. Find the Download page and click the download button that matches your operating system.

This downloads the Ollama installation script.

3. Go to the directory where you downloaded the install script.
4. Run the installation using the following methods depending on your operating system:
    
    * For Linux
   
        chmod +x ollama_linux.sh  
        ./ollama_linux.sh
    
     * For macOS
   
         chmod +x ollama_macos.sh 
        ./ollama_macos.sh

    * For Windows

    1. Double-click the download script and follow the on-screen instructions.

## Next steps
When the installation is complete, configure Ollama and pull and run your first model.
